## Acceptance Procedure

1. `/opt/homebrew/bin/rtk just storybook-release-acceptance-artifacts` を実行し、review artifact、source screenshot、headless live-acceptance artifact を最新化する。

5. `/opt/homebrew/bin/rtk just storybook` を起動し、KUC 実部品 viewer を実機で操作する。smoke / screenshot / score gate の通過だけで accepted にしない。
6. Acceptance Checklist と Automated Evidence Matrix の各行について、表示、操作、性能、KatanA viewer / export HTML / export PDF との比較結果を実操作で確認する。
